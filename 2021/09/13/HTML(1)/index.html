4.图像标签

常见的图像格式

  • jpg
  • gif
  • png
  • bmp

图像标签格式

语法：

src 、alt、title 被称为属性，多个属性之间用空格隔开，一个标签里面可以有多个属性

对于图像而言，有两个必须要写的属性，即 src 和 alt

  • src : 图片路径
  • alt : 当图片不能正常显示的时候，alt会在图片应该显示的位置显示对应的文字
  • title： 鼠标悬停提示文字
  • width ：图片宽度
  • height ：图片高度
