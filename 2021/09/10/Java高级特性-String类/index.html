Java高级特性—String类

String类

String类概述

在Java中，字符串被当做String类型的对象来处理。String类位于java.lang包中，默认情况下，该包被自动导入所有程序。创建String对象的方法如下代码所示

String s = “Hello World”;

或者

Stirng s = new String(“Hello World”);

String类提供了许多有用的方法，例如获取字符串长度、对两个字符串进行比较、连接两个字符串以及提取一个字符串中的某一部分等。可以用String类提供的方法来完成对字符串的操作。

下面介绍String的常用方法

方法 作用
length(); 返回字符串的长度
equals(); 比较两个字符串对象的内容是否一致，返回值为布尔类型，如果相同则为true，不相同为false
equalsIgnoreCase(); 忽略大小写比较字符串1和字符串2，如果相同，返回true；不相同返回false
toLowerCase(); 转换字符串中的英文字符为小写
toUpperCase(); 转换字符串中的英文字符为大写
trim(); 截取字符串前后的空格后返回新的字符串

常用的提取和搜索字符串的方法

方法 作用
public int indexOf(int ch) 搜索并返回第一个出现字符ch的位置，没找到返回-1
Public int indexOf(String value) 搜索并返回第一个出现字符串value的位置，没找到返回-1
public int lastIndexOf(int ch) 搜索并返回最后一个出现字符ch的位置
public int lastIndexOf(String value) 搜索并返回最后一个出现字符value的位置
public String SubString(int index) 提取从指定索引位置开始的部分字符串
public String SubString(int beginindex,int endindex) 提取beginindex和endindex之间的 字符串
