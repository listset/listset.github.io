发表于2021-09-10更新于2021-08-18

包装类

包装类概述

Java语言是面向对象的，但是Java中的基本数据类型却不是面向对象的，这在实际开发中存在很多的不便

为了解决这个不足，在设计类时为每个基本数据类型设计了一个对应的类，称为包装类。

包装类均位于 Java.lang 包中
包装类提供了一系列实用方法
集合不允许存放基本数据类型数据，存放数字时，要用包装类型

Byte、Short、Integer、Long、Float、Double 都是Number类型

包装类和基本数据类型的对应关系如下

基本数据类型 包装类型
byte Byte
boolean Boolean
short Short
char Character
int Integer
long Long
float Float
double Douoble

包装类的两个用途

包装类作为基本数据类型对应的类存在，方便对象操作
包装类包含每种基本数据类型的相关属性，如最大值、最小值等，以及相关的操作方法

什么是装箱和拆箱？

在JavaSE 5.0版本后程序员不需要编码实现基数据类型和包装类之间的转换，编译器会自动完成
