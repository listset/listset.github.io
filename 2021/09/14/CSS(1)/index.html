1.2 CSS的概念

  • Cascading Style Sheet 级联样式表
  • 表现HTML文件样式的计算机语言
    • 样式定义如何显示HTML元素
    • 包括对字体、颜色、边距、高度、宽度、背景图片、网页定位等设定

1.3 CSS的优势

  • 内容与表现分离
  • 网页的表现统一，容易修改
  • 丰富的样式，是的页面布局更加灵活
  • 减少网页的代码量，增加网页的浏览速度，节省网络宽带
  • 运用独立于页面的CSS，有利于网页被搜索引擎收录
