HTML

1.列表

什么是列表

列表就是信息资源的一种展示形式。它可以使信息结构化和条理化，并以列表的形样式显示出来，以便浏览者能更快捷地获得相应的信息

列表的分类

  • 无序列表
  • 有序列表
  • 定义列表

HTML 列表标签
| 标签 | 描述 |
| :—-: | :—-: |
| <ol> | 定义有序列表 |
| <ul> | 定义无序列表 |
| <li> | 定义列表项 |
| <dl> | 定义列表 |
| <dt> | 自定义列表项目 |
| <dd> | 定义自定列表项的描述 |

1.1无序列表

示例：

  • aaa
  • bbb

ul 声明无序列表 li 声明列表项
ul 默认的是黑心小圆点 ：disc ；circle：空心圆圈 ；square：黑心小方块

特性：

没有顺序，每个<li>标签独占一行
默认<li>标签项前面有个实心小圆点
一般用于无序类型的列表，如导航、侧边栏新闻、有规律的图文组合模块等

1.2有序列表

示例：

  1. aaa
  2. bbb

ol 声明有序列表 li 声明列表项
ol 默认是有序数字 1 还有：A a I i 四种取值

特性：

有顺序，每个<oi>标签独占一行
默认<oi>标签项前面有顺序标记
一般用于排序类型的列表，如试、问候卷等

1.3定义列表

示例：
volume

水果
苹果
桃子
李子

dl：声明定义列表，dt：声明列表项，dd：定义列表内容

特性：

没有顺序，每个<dt>标签、<dd>标签独占一行
默认没有标记
一般用于一个标题下有一个或多个列表项的情况

table：表格标签；tr：行标签；th：标题标签；td：单元格标签
colspan=”2”：表示单元格跨两列 ；rowspan=”2”：表示单元格跨两行

3.1 input

语法：

type：input元素类型；name：input元素名称；value：input元素的值

属性 说明
type 指定袁术的类型。text、password、checkbox、radio、submit、reset、file、hidden、image和button，默认为text
name 指定表单元素的名称
value 元素的初始值，type为radio是必须指定的一个值
size 指定表单元素的初始宽度，当type为text或password时，表单元素的大小以字符为单位。对于其他类型，宽度以像素为单位
maxlength type为text或password时，可输入的最大字符数
checked type为radio或checkbox时，指定按钮是否被选中
