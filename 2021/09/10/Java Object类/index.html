Object类介绍

Object 是所有的类的直接或间接父类
如:Cat后面的 extends Object 不写也默认继承

Object类被子类经常重写的方法

方法 说明
toString() 比较两个对象是否是同一个对象，是则返回true
equals() 单元格
hashCode() 返回该对象的哈希代码值
getClass() 获取当前对象所属的类信息，返回Class对象
